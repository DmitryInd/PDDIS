Разработать два ansible playbook:
1. Поднимаем на двух серверах веб-сервер, запускает его (например, nginx). 
2. Деплоит на два сервера ваше приложение с shell скриптом его запуск. Деплой означает: проверка конфигурации сервера, установка всех необходимых компонентов, распаковка/конфигурация вашего приложения и инструмент запуска


Если у вас домашний компьютер на Windows вы не сможете разрабатывать playbook (Windows is not supported for the control node, read more about this in Matt Davis’s blog post.).
Поэтому, используйте одну из машин (например, CentOS) в качестве control node. В файле host пропишите обе машины по IP, ansible в этом случае будет поднимать две ssh сессии на обе машины, несмотря на то что одна из них для него localhost
